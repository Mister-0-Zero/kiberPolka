# Data Fusion Contest 2026 — Киберполка
## Базовый README проекта

## Цель проекта

Решить задачу **multi-label классификации** для 41 финансового продукта клиентов банка на основе анонимизированных табличных данных.

Каждому клиенту необходимо предсказать вероятность открытия каждого из продуктов.

Метрика соревнования:

**Macro ROC-AUC**

То есть среднее значение ROC-AUC по всем 41 таргету.

Важно:

- каждый таргет имеет одинаковый вес
- редкие классы влияют на итоговую метрику так же, как частые
- стабильность по всем таргетам критична

### Метрика
